Personal Projects

VR Spline Rider

Tech stack: Unity 2.0f(C#), 5UDE

VR Spline Rider is an application that allows a user to build and ride splines in VR. The user creates control points using their right hand, which are used to automatically generate Catmull-Rom splines. The control points can be modified or deleted by the user in real time. The user can also ride their spline (scaled to the real world) by clicking the "MENU" button. We dynamically calculated the quaternions of control points and used them to rotate the user through the spline accordingly. We also implemented our own physics system that emulated real life physics.

Color Compressor

Tech stack: Java, JPanel

Color Compressor is an open source project that uses an unsupervised learning algorithm known as K-means in order to compress images into a user given number of colors. Users can also see the color palette chosen by the algorithm and even change these colors to their liking. This simple GUI also gives a user the option to load, save, and undo/redo (CTRL+Z/CTRL+Y) palette changes.
